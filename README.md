# 🚢 Titanic Survival Prediction
Machine learning model to predict Titanic passenger survival
## 📌 Project Overview
This project predicts whether a passenger survived the Titanic disaster using Machine Learning.

## 🧠 Model Used
- Logistic Regression

## 📊 Features Used
- Age
- Gender
- Passenger Class
- Fare
- Family details
- Embarked

## ⚙️ Steps Performed
1. Data Loading
2. Data Cleaning
3. Feature Engineering
4. Model Training
5. Model Evaluation

## 📈 Results
Achieved ~80% accuracy.

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib & Seaborn
